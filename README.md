BarCode Coder Library (BCC Library)
=======================
BCC generates some kinds of barcodes for php GD and FPDF.

Managed barcodes
--------------
* standard 2 of 5 (std25)
* interleaved 2 of 5 (int25)
* ean 8 (ean8)
* ean 13 (ean13)   
* code 11 (code11)
* code 39 (code39)
* code 93 (code93)
* code 128 (code128)  
* codabar (codabar)
* msi (msi)
* datamatrix (datamatrix)
